Um aplicativo de linha de comando para gerenciar tarefas, feito para praticar Python e GitHub.

## Como usar

- Adicionar tarefa:
  ```bash
  python projeto.py add "Minha tarefa"
  ```
- Listar tarefas:
  ```bash
  python projeto.py list
  ```
- Marcar tarefa como concluÃ­da:
  ```bash
  python projeto.py done 1
  ```
- Remover tarefa:
  ```bash
  python projeto.py remove 1
  ```
- Limpar todas as tarefas:
  ```bash
  python projeto.py clear
  ```

> O arquivo `todos.json` guarda suas tarefas automaticamente.
